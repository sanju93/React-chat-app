## Project Name : Chat-App with React

## Prequisite
1. install npm
2. install react
3. FireStore database

## How to Start the Project
1. Git Clone `https://github.com/sanju93/React-chat-app`
2. npm install
3. npm start
4. From Url `https://chat-app-6cde5.web.app/`
